# Templates Directory

This directory contains templates for building all harness materials.

Each template includes:
- Purpose of the material
- Who uses it (which agents)
- Key sections to include
- Examples or references
- Quality criteria

## How To Use Templates

1. Read the template for your assigned material
2. Follow the structure provided
3. Fill in all sections
4. Include examples where indicated
5. Save completed material to `../materials/` directory
6. Update CURRENT_STATUS.md and TASKS.md
