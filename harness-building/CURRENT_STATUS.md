## How To Update This File

**When you start a task:**
1. Move it from "Up Next" to "In Progress"
2. Add your identifier (e.g., "Gemini Agent - Dec 15")
3. Update "Last Updated" timestamp at top
4. Commit to GitHub

**When you complete a task:**
1. Move it from "In Progress" to "Recently Completed"
2. Add completion timestamp
3. Update progress percentage if applicable
4. Check off in TASKS.md too
5. Update "Last Updated" timestamp at top
6. Commit to GitHub

**When you encounter a blocker:**
1. Add to "Blockers / Issues" section
2. Describe the issue clearly
3. Note what's needed to unblock
4. Update "Last Updated" timestamp at top
5. Commit to GitHub
